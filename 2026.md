### 📌 자주 헷갈리는 분기

#### 내가 서버? 클라이언트?

```
"URI : POST http://127.0.0.1:8080/..."
    ↑ 이 8080이 내 프로그램 포트면 → 나는 SERVER (Jetty Server)
    ↑ 이 8080이 다른 프로그램 포트면 → 나는 CLIENT (HttpClient)

문제에 "MOCK이 호출함" → 나는 SERVER
문제에 "MOCK으로부터 응답 받음" → 나는 CLIENT
문제에 "HTTP로 변경" + 소문항3 → 보통 SERVER
문제에 "외부 시스템 호출" / "AI 모델 호출" → CLIENT 추가
```

#### List vs Map vs Queue?

```
순서 유지 + 인덱스 접근       → List
key로 빠른 조회             → Map
FIFO (먼저 들어간 게 먼저)  → Queue (LinkedList)
중복 없음 + 포함 여부만 체크 → Set
이름 있는 여러 개            → Map<String, List/Queue/...>
```

#### Worker(라이브러리 제공)? 직접 구현?

```
"Worker는 라이브러리로 제공" → 직접 만들지 말고 그냥 호출만
"removeExpiredStoreItems 구현" → Worker.java 수정 (제공 클래스 안에 메서드만 추가)
직접 구현 + 호출 → 직접 만들기
```

### Pattern B: HTTP 그대로 전달 (2022 Service Proxy)

> 받은 Method/Path/Query/Body 를 그대로 다른 서버로 전달

```java
static class ProxyHandler extends AbstractHandler {
    List<Route> routes;       // [{pathPrefix, url}, ...]
    HttpClient httpClient;
    
    @Override
    public void handle(String target, Request baseRequest,
                       HttpServletRequest req, HttpServletResponse resp)
                       throws IOException, ServletException {
        
        String path = req.getRequestURI();
        String query = req.getQueryString();
        String method = req.getMethod();
        
        // 라우팅: pathPrefix 매칭
        Route matched = null;
        for (Route r : routes) {
            if (path.startsWith(r.pathPrefix)) {
                matched = r;
                break;
            }
        }
        if (matched == null) {
            resp.setStatus(404);
            baseRequest.setHandled(true);
            return;
        }
        
        // 전달할 URL = 매칭된 url + path + query
        String forwardUrl = matched.url + path;
        if (query != null) forwardUrl += "?" + query;
        
        try {
            org.eclipse.jetty.client.api.Request fwd = httpClient.newRequest(forwardUrl)
                .method(HttpMethod.fromString(method));
            
            // x- 로 시작하는 헤더 그대로 전달
            Enumeration<String> headerNames = req.getHeaderNames();
            while (headerNames.hasMoreElements()) {
                String h = headerNames.nextElement();
                if (h.toLowerCase().startsWith("x-")) {
                    fwd.header(h, req.getHeader(h));
                }
            }
            
            // POST면 바디 전달
            if (method.equals("POST")) {
                String body = req.getReader().lines().reduce("", (a, c) -> a + c);
                String contentType = req.getContentType();
                fwd.content(new StringContentProvider(body), contentType);
            }
            
            ContentResponse cr = fwd.send();
            
            // 응답 그대로 전달
            resp.setStatus(cr.getStatus());
            String ct = cr.getMediaType();
            if (ct != null) resp.setContentType(ct);
            resp.getWriter().write(cr.getContentAsString());
            
        } catch (Exception ex) {
            ex.printStackTrace();
            resp.setStatus(500);
        }
        baseRequest.setHandled(true);
    }
}

class Route {
    String pathPrefix;
    String url;
}
```

#### 소문항2 풀이 (시간 범위 필터)

```java
// 위 코드 + 시간 범위 콘솔 입력 + 필터링
Map<String, String[]> predictMap = new HashMap<>();   // [ts, value]
Map<String, String>   actualMap  = new HashMap<>();

// 파일 읽기 (timestamp도 저장)
BufferedReader br = new BufferedReader(new FileReader("MONITORING.TXT"));
String line;
while ((line = br.readLine()) != null) {
    if (line.trim().isEmpty()) continue;
    String[] parts = line.split("#");
    String reqId = parts[0];
    String ts    = parts[1];
    String type  = parts[2];
    String value = parts[3];
    
    if (type.equals("P"))      predictMap.put(reqId, new String[]{ts, value});
    else if (type.equals("A")) actualMap.put(reqId, value);
}
br.close();

// 콘솔 입력 - 시간 범위 (예: "2025041010")
Scanner sc = new Scanner(System.in);
String timeWindow = sc.nextLine().trim();

int total = 0, correct = 0;
for (Map.Entry<String, String[]> e : predictMap.entrySet()) {
    String reqId = e.getKey();
    String ts    = e.getValue()[0];
    String pVal  = e.getValue()[1];
    
    if (!ts.substring(0, 10).equals(timeWindow)) continue;
    
    if (actualMap.containsKey(reqId)) {
        total++;
        if (pVal.equals(actualMap.get(reqId))) correct++;
    }
}

System.out.println(correct + "/" + total);
```

### "HTTP가 동작 안 함"

| 증상 | 원인 | 해결 |
|------|------|------|
| 응답이 안 옴 | `setHandled(true)` 누락 | 핸들러 끝에 추가 |
| Connection refused | 서버 안 떠있음 | server.start() 호출 확인 |
| 포트 충돌 | 이미 다른 프로세스 사용 | `netstat -ano | findstr 8080` 으로 확인 후 kill |
| HttpClient 에러 | start() 안 함 | `httpClient.start()` 추가 |
| 응답 body 비었음 | getWriter().write 후 close 됨 | flush() 추가 또는 setHandled 위치 |
